# RecoverAI-Lite Frontend
> Patient-Centric Recovery Interface

A lightweight, high-performance React application focused exclusively on the Clinical Companion.

## 🚀 Features
- **Secure Unified Login**: Direct access to your clinical recovery environment.
- **Clinical Memory**: Voice-first logging and clinical history viewer.
- **Responsive Design**: Optimized for tablets and laptops in recovery environments.

## 🛠️ Tech Stack
- **Framework**: React 19 + Vite.
- **Styling**: Tailwind CSS + Framer Motion.
- **State**: Zustand.

## 🛠️ Setup
1. `npm install`
2. `npm run dev` (Port 5173)
