# RecoverAI Clinical Companion
> The Caregiver OS

A voice-first clinical logging system that transforms natural patient/caregiver dialogue into structured medical data.

## 🚀 Features
- **Voice Intelligence**: Powered by Google Gemini for high-accuracy medical entity extraction.
- **Protocol Adherence**: Designed to follow clinical discharge instructions precisely.
- **Visual Log Viewer**: Clean, intuitive history of all medications and symptoms.

## 🛠️ Tech Stack
- React 18+ (bundled as a module).
- Google Generative AI SDK.
- Local Clinical Memory persistence.
